# Basic Definitions
- eight different built-in types of data in java
- Characters and Strings
- Integers
- Real numbers
- Booleans
- Comparisons
- Type Conversions

# Characters and Strings
1. char
  - alphanumeric character or symbol
    
2. String
  - a sequence of characters
  - most common operation: concatenation

```
String a, b, c;
a = "Hello,";
b = " Bob";
c = a + b;
```
